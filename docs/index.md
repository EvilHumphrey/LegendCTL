# LegendCTL

A free, open-source (MIT), fully-local **Windows** app for configuring the **ZD Ultimate Legend** controller —
deadzones, sensitivity curves, polling rate, button mapping, triggers, vibration, lighting, and profiles. It runs
**standalone** (no official ZD app required), talks to the controller over USB-HID, makes **zero network calls**, and
has no telemetry, drivers, macros, or background service.
