# Community references & adoption

A running, public record of where LegendCTL has been independently mentioned,
reviewed, discussed, or used in the wild. It exists for two reasons:

1. **Transparency** — so anyone can see the project's real footprint, not a
   marketing claim.
2. **Eligibility evidence** — the free [SignPath Foundation](https://signpath.org)
   code-signing program for open-source projects asks for external verification
   signals (community adoption, independent references/discussions, articles)
   before accepting a project. This page collects those signals as they appear,
   to support a future application. (See the [code-signing policy](code-signing-policy.md).)

This is an honest, early-stage log: the project is new, so the lists below are
short and will grow over time. Entries are independent third-party references —
not the maintainer's own posts.
